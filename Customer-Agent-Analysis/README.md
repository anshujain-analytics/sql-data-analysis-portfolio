# 👥 Customer & Agent Business Analysis using SQL

## 📌 Project Overview

This project analyzes customer and agent data using SQL to evaluate agent performance, customer outstanding balances, customer distribution, and business relationships.

The project demonstrates advanced SQL techniques including joins, aggregate functions, subqueries, and business-oriented reporting.

---

## 📂 Dataset

The database consists of two relational tables:

- AGENTS
- CUSTOMER

The tables are created and populated using SQL scripts included in this repository.
