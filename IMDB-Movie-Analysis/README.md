# 🎬 IMDB Movie Analysis using SQL

## 📌 Project Overview

This project analyzes the IMDB Movie dataset using SQL to uncover insights into movie ratings, director performance, genre-wise revenue, and yearly top-rated films.

The project demonstrates advanced SQL techniques including window functions, aggregate analysis, and correlated subqueries.

---

## 📂 Dataset

- IMDB-Movie-Data.csv

**SQL Concepts**

**SQL Concepts**
