# ⚾ Baseball Player Analysis using SQL

## 📌 Project Overview

This project analyzes baseball player statistics using SQL. The objective is to identify player characteristics by applying filtering, aggregate functions, sorting, and subqueries.

---

## 📂 Dataset

- baseball_data.csv

---

## 🎯 Business Questions

### Q1. Who are the top 5 players having height greater than the average height?

**SQL Concepts**
- Aggregate Function (AVG)
- Subquery
- ORDER BY
- LIMIT

---

### Q2. Find the bottom 2 left-handed players with the least weight.

**SQL Concepts**
