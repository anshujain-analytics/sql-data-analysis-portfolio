# 🎵 Chinook Music Store Analysis using SQL

## 📌 Project Overview

This project analyzes the Chinook Music Store database using SQL to uncover customer purchasing behavior, music genre popularity, artist performance, and sales trends.

The analysis demonstrates the use of joins, aggregate functions, grouping, and business-oriented SQL queries to generate meaningful insights.
